# 광고 단위 슬롯 ID 교체 안내

각 가이드에 반응형 디스플레이 광고 단위가 배치돼 있습니다(상단·하단 2개).
현재 슬롯 ID는 플레이스홀더(`data-ad-slot="0000000000"`)입니다.

## 애드센스 승인 후
1. AdSense → 광고 → 광고 단위별 → **디스플레이 광고**(반응형) 생성 → 슬롯 ID 복사
2. 아래 명령으로 일괄 교체(제가 대신 해드릴 수 있음):
   find guides -name "*.html" -exec sed -i '' 's/data-ad-slot="0000000000"/data-ad-slot="<실제슬롯ID>"/g' {} +
3. git push → 반영

## 더 쉬운 대안: 자동 광고(Auto Ads)
AdSense → 광고 → 사이트별 → **자동 광고 켜기**. 슬롯 없이 페이지에 자동 배치됩니다
(계정 스크립트가 이미 전 페이지에 삽입돼 있어 바로 작동).
